# Space Invaders

## Popis
Tento projekt je kopie klasické hry Space Invaders. Projekt je napsán v herním enginu Godot. Hra obsahuje základní mechaniky, jako je pohyb hráče, ale není kompletně dokončena.

## Aktuální stav
- Hráč může ovládat loď
- Nepřátelé se hýbají
- Home screen

## Absence prvků
- Střílení z laseru
- Bunkery
- Skóre
